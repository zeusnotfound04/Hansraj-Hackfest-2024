# AI Voice Assistant

This project is a simple AI voice assistant built using Python. The assistant listens to your commands and types them out, as well as executing predefined commands like opening a web browser or Notepad.

## Features

- **Voice Recognition**: Understands user commands through speech.
- **Text-to-Speech**: Converts text responses to speech.
- **Typing Automation**: Types out any commands that aren't predefined actions.
- **Custom Commands**: Execute specific commands such as opening a browser or searching the web.

## Requirements

- Python 3.x
- Libraries:
  - `SpeechRecognition`
  - `PyAudio`
  - `pyautogui`
  - `keyboard`
  - `pyttsx3`

You can install the required libraries using pip:

pip install -r requirements.txt
